# Safety Policy - Domu Match

## Overview

Domu Match is committed to providing a safe and secure platform for students to find compatible roommates. This document outlines our safety policies, reporting procedures, and user rights.

## Reporting Process

### How to Report

Users can report other users or messages through the following methods:

1. **In-Chat Reporting**: Click the report button in any chat conversation
2. **Match Reporting**: Report users from match suggestion cards
3. **Categories Available**:
   - Spam/Fake accounts
   - Harassment
   - Inappropriate content
   - Other safety concerns

### What Happens After Reporting

1. **Immediate Actions**:
   - Report is logged in our system
   - Admin team is notified
   - Report is queued for review

2. **Auto-Blocking**:
   - Users with 3+ reports in 24 hours are automatically blocked
   - Blocked users cannot match with others or send messages

3. **Admin Review**:
   - Reports are reviewed by administrators
   - Actions may include:
     - Warning the user
     - Suspending the account
     - Dismissing the report if unfounded

### Rate Limiting

- Users can submit up to 5 reports per hour
- This prevents abuse of the reporting system

## Blocking Policy

### User Blocking

Users can block other users at any time:

- **From Match Cards**: Click "Block User" on any match suggestion
- **From Chat**: Use the block option in chat settings
- **Effects of Blocking**:
  - Blocked users cannot match with you
  - Messages from blocked users are hidden
  - You cannot send messages to blocked users

### Platform Blocking

The platform may automatically block users who:

- Receive 3+ reports within 24 hours
- Violate community guidelines repeatedly
- Engage in spam or fraudulent activity

## Data Retention

### Verification Data

- KYC provider data is retained according to provider policies (typically 90 days)
- Verification status is stored indefinitely for platform safety
- Users can request deletion of verification data per GDPR

### Chat Messages

- Messages are stored for the duration of the chat
- Deleted chats remove all associated messages
- Admin exports may be retained for safety investigations

### Reports

- Reports are retained for 1 year after resolution
- After 1 year, reports are anonymized and archived
- Critical safety reports may be retained longer for legal compliance

## GDPR Compliance

### User Rights

Users have the right to:

1. **Access**: Request a copy of all personal data we hold
2. **Rectification**: Correct inaccurate personal data
3. **Erasure**: Request deletion of personal data (subject to legal obligations)
4. **Portability**: Receive data in a machine-readable format
5. **Object**: Object to processing of personal data
6. **Restriction**: Request restriction of data processing

### Data Processing

- Personal data is processed only for platform functionality
- Verification data is processed by third-party KYC providers (Veriff/Persona/Onfido)
- Users consent to data processing during account creation

### Data Security

- All data is encrypted in transit (HTTPS/TLS)
- Sensitive data is encrypted at rest
- Access to personal data is restricted to authorized personnel only

## Safety Features

### Verification System

- All users must verify their identity before matching
- Verification uses industry-standard KYC providers
- Verification status is visible to other users

### Message Moderation

- Messages are automatically filtered for:
  - Links (blocked for safety)
  - Email addresses (blocked for privacy)
  - Phone numbers (blocked for privacy)
- Violations result in message rejection

### Admin Oversight

- All reports are reviewed by administrators
- Admin actions are logged for accountability
- Users can appeal admin decisions

## Contact

For safety concerns or GDPR requests:

- **Email**: info@domumatch.com
- **Response Time**: Within 48 hours for urgent safety issues
- **GDPR Requests**: Processed within 30 days

## Updates

This policy may be updated periodically. Users will be notified of significant changes via email or platform notification.
